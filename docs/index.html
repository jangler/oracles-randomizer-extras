Oracles Randomizer Extras

This website shuffles music and sound effects in a Zelda: Oracle of Ages/Seasons ROM.

Only US versions have been tested. Randomized ROMs are supported as long as they haven't been recompiled from the disassembly (like Drenn's cross-items fork does).

Enemies are replaced only in ways that cannot block randomizer logic.

Other features may be added in the future!
